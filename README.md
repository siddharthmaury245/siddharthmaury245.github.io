## ğŸ› ï¸ How It Works

Using VORTEX SUITE is as easy as **1-2-3**:

1.  **Select Apps:** Browse through categories (Browsers, Dev, Gaming, etc.) and click on the apps you want.
2.  **Generate Script:** Click the **"INSTALL APPS"** button to download the `Vortex_Installer.bat` file.
3.  **Run & Relax:** Open the file on your PC. The script will automatically download and install the latest versions of your selected software.
